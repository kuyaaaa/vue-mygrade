# vue-mygrade

> 个人考试成绩记录系统

## 简单说明
本demo使用json-server做为模拟后端数据，前端运用 vue + vuex + axios + router + elementui + echarts 制作。

具体功能分为学习计划、成绩增删改查和一个简单的显示最近日期的12次考试数据分科目成绩折线图。

## Dependencies
``` bash
"dependencies": {
    "axios": "^0.19.2",
    "concurrently": "^5.1.0",
    "echarts": "^4.6.0",
    "element-ui": "^2.13.0",
    "vue": "^2.5.2",
    "vue-router": "^3.0.1",
    "vuex": "^3.1.2"
  }
```

## Build Setup

``` bash
# install dependencies
npm install

# json-server 安装
npm install -g json-server

# nodemon 安装
npm install -g nodemon

#json-server 启动 localhost:3000
npm run server

#前端 启动 localhost:8080
npm run start

# 同时启动
npm run dev
```
